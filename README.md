# Dessert Shop
使用Vue.js打造的甜點商店。使用者可將喜歡的甜點加入購物車，註冊或登入後可以建立訂單、查詢訂單
> 後端佈署於heroku，初次使用需等待伺服器啟動

[Demo](https://wutiramisu.github.io/dessert-shop/)

## 測試帳號
* username: user
* password: user

## 功能介紹
1. 會員註冊/登入
2. 購物車數量增減、刪除
3. 建立、查詢訂單

## 使用技術
* Vue.js
* vue-router
* Vuex
* SCSS
* RWD Design
* 第三方套件使用：vue-carousel
* 後端使用Express、MongoDB搭建簡易伺服器

## 聲明
本作品僅供練習使用，不做任何商業用途
